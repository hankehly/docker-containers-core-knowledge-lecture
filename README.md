# docker-containers-core-knowledge-lecture
Demonstration code for [lecture on docker containers](https://enechange-meetup.connpass.com/event/271139/).

### コンテナ開発の流れ
1. プログラムを作る (ruby, js, python, php, bashなど)
2. Dockerfileを作成する
3. コンテナイメージをビルドする
4. コンテナを起動する
5. コンテナをレジストリーにpushする
